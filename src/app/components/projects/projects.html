Room Reservation System

Responsive Room Booking Platform • HTML, CSS, JavaScript, Bootstrap

View Live Demo

Built a fully functional, modern room booking website that allows users to search, filter, and reserve rooms with a seamless cart-based booking flow — delivering an intuitive and responsive user experience.

Key Features Implemented:
  • ✅ Advanced search & filter by price, location, and amenities
  • ✅ Dynamic room details with image gallery and specifications
  • ✅ Add to cart with real-time quantity and price calculation
  • ✅ Fully responsive design (mobile + desktop)
  • ✅ Smooth cart management and booking summary
  • ✅ Clean, modern UI using Bootstrap 5
Technologies: HTML5 • CSS3 • Vanilla JavaScript • Bootstrap 5

A foundational project that strengthened my frontend architecture, state management, and responsive design skills.
